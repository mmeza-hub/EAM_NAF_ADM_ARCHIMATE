# EAM_NAF_ADM_ARCHIMATE
Repository für die strukturierte Entwicklung, Dokumentation und Automatisierung von Enterprise-Architekturen auf Basis von TOGAF ADM, NAF v4 und ArchiMate. Es dient als zentrale Grundlage für Architekturmodelle, Metamodelle, Viewpoints, Governance-Artefakte und wiederverwendbare Architekturbausteine im Unternehmens- und Verteidigungsumfeld.
